## Description
A simple implementation of Linux Shell using C. 
Takes input from the user, processes it and takes action accordingly.
Set of Implemented built-in functionalities:
```
"cd"
"history"
"clear"
"help"
"exit"
```
## Prerequisites

1. Linux/unix system
2. C compiler

## Execution commands

```
gcc -o myshell myshell.c

./myshell

```
